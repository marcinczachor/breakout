Breakout Game

Description

The game consist of ball, paddle and wall of bricks. A player has 3 attempts to accomplish the current level. If a player aquires 150 points, the next level is set and ball starts to move faster. Also a player gains one additional life. The game finishes when a player reaches max level, which is currently set to 3 or loose all his available lives.

Controls

  • Move paddle to the left by clicking ⇦ (left arrow).
  • Move paddle to the right by clicking ⇨ (right arrow).
